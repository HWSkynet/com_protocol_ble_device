## 协议特性

- 低成本（即便是运行速度很慢的MCU，比如低至16bits 4M的配置，也能够轻松的解析和封装数据包）
- 高可靠（端到端确认）
- 兼容性（完整的向后兼容）
- 统一性（标准格式，统一解析）
- 可组网（提供轻量级路由功能，可在由BLE设备组成的小型网络中通信）
- 高效率（原生支持全异步的端到端确认，不浪费任何一个通信帧）
